Cover Letter

I recently graduated from the University of Waterloo in Management and Data Sciences as a master’s student.

My master's thesis is about Data Science, Machine Learning (ML), and Artificial Intelligence (AI).

Its title is “Data-Driven Patient Scheduling with AI and Reinforcement Learning.”

The main goal of that is to infer the intentions used by the decision-makers, who schedule patients, based on their historical decisions. To achieve this, I first developed a mathematical model that captures the complexities of scheduling (Data Modelling).

To check the robustness of the method, I implemented a simulation and built a AI-based patient scheduler to test hypotheses, improving the decision-makeing process and total waiting times by 30%. (Predictive AI, Data Science).

Furthermore, I analyzed Canadian hospitals datasets using Python and Pandas, and successfully estimated the cost function. I also predicted the next states of the system using predictive analytics and AI. I worked with SQL as a database for my project (SQL, ML, Data Analytics).

This thesis showcased my Quantitative Data Analysis, Machine Learning skills, AI, Model Optimization, Predictive Analytics, Forecasting, Data Visualization, and Git.

Additionally, I earned IBM's Machine Learning with Python Certificate, learning some ML techniques and using Python libraries such as TensorFlow, PyTorch, and Pandas.

I also built a Scalable Machine Learning API, enabling non-expert users to upload datasets, train models, save best-performing models, and receive predictions and evaluation scores. I used Docker files to ensure seamless local deployment and developed Kubernetes deployment and service YAML files.
